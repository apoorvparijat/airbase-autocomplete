## Installation

`pip install -r requirements.txt`

## Start Server

In the root folder, run

`python app.py`

Server is started at host=127.0.0.1,port=5000

## Example Queries

-   http://localhost:5000?q=hello
-   http://localhost:5000?q=cat&n=5
-   http://localhost:5000?q=eryx&n=1000
